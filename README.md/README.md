# Mi Proyecto Web - Fernando Montero Gómez
### Grado en Ingeniería Informática - UFV

## Descripción
Este sitio web es un proyecto individual que presenta mi perfil personal, mi formación académica y mi red de contactos.

## Páginas Incluidas
* **Inicio**: Presentación general.
* **Temática**: Información detallada sobre el Pádel.
* **Acerca de mí**: Perfil personal y pasiones.
* **Grado**: Información sobre la UFV y mi carrera.
* **Red**: Conexiones con compañeros.
* **Contacto**: Formulario de comunicación.

## Tecnologías utilizadas
* HTML5
* CSS3 (Flexbox y Grid)
